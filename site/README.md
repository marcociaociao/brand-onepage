# brand-onepage /site
One-page scura, narrativa, con 3 capitoli demo.
Stack: Next.js 14 + TypeScript + Tailwind + Framer Motion.

## Sviluppo locale
1) cd site
2) npm i
3) npm run dev
Apri http://localhost:3000
